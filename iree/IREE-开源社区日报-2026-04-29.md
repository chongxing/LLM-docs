## 🏷️ 技术方向洞察

### 1. HAL/CPU Local Runtime 重写 (#24248)
- **benvanik** 的重大架构更新，从近地面重写了 CPU 本地运行时
- 引入了详细的程序化反馈机制用于调优不同工作负载和目标
- 这是 IREE CPU 后端的里程碑式更新，称为 "v3"

### 2. HAL Capture API 与 Replay 工具 (#24288)
- 新增 HAL 捕获 API 和重放工具，用于调试和性能分析
- 有助于捕获和重现 GPU/加速器执行轨迹

### 3. CPU Data-Tiled 编码到 inner_tiled (#24289)
- **bjacob** 推进 CPU 后端的 data-tiled matmul 优化
- 将 encodings 物化为 `inner_tiled` 而非 `linalg.mmt4d`
- 包含基于成本模型的 intrinsic 选择

### 4. DispatchCreation 持续优化
- **#24293**: 为低并行度目标 (如 RDNA GPU) 添加 split reduction 参数集
- **#24166**: 标量 reduction 与并行消费者的融合优化
- **#24271**: Attention K1/N 单位维度保留 (已合并)

### 5. AMDGPU 后端进展
- **#24247**: ROCDL buffer load 分组优化，改善指令调度
- **#24287**: vector interleave/deinterleave 合法化到一维向量

### 6. LLVM 同步
- **#24284**: 升级到 `llvm-project@fd761415a5c`，保持与上游同步
