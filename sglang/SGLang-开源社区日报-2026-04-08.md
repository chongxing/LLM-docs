## 💡 Technical Insights

### 1. Linear Attention Backend Registration API
**PR #21983** introduces a registration API for external linear attention backends, allowing hybrid models (GDN, KDA, Mamba2, Lightning) to integrate without modifying SGLang core files. This is a significant architectural improvement for extensibility.

### 2. Performance Optimization
**PR #21771** restores `torch.compile` fusion for topk postprocessing that was inadvertently removed in PR #16945. This improves performance in expert-parallel / EPLB paths by fusing kernels instead of running them separately.

### 3. NVIDIA FP4 Support
Commit `7546d04` enables FP4 flashinfer trtllm routed MoE support, expanding quantization capabilities for Blackwell GPUs.

### 4. DFLASH Speculative Decoding
Commit `f08726f` adds DFLASH speculative decoding support, a major feature for improving inference throughput.

### 5. NSA Indexer Optimization
Commit `671fe73` reduces unnecessary kernels and copies in the NSA (Natively Trainable Sparse Attention) indexer, improving efficiency.
