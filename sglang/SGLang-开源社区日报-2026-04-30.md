## 🔍 重点技术方向

### 1. DeepSeek-V4 支持持续完善
- **#24104** - 为 DeepSeek-V4 cookbook 的 Basic Configuration 章节添加 12 个手动 AIME25 测试（覆盖 B200/H200 × Flash/Pro × Low-Latency/Balanced/Max-Throughput 所有组合）
- **#24035** / **#23990** - 清理 DeepSeek V4 相关的错误注释和备份配置
- **#24050** - AMD DeepSeek V4 集成第 5 步（indexer TilelangAttn）
- **#24099** - 将模型缓存同步屏障迁移到平台操作（DO NOT MERGE）

### 2. FA3 确定性推理优化
- **#24101** - 在 FA3 确定性推理模式下启用 `num_splits=4`（原为强制 num_splits=1），通过 `batch_invariant` 修复保持位精确确定性。短 prefill 吞吐量提升 **2.1x**，E2E 延迟降低 **2.25x**

### 3. CI / 基础设施改进
- **#24102** - 为 runner-utilization 报告添加 per-host 利用率视图（解决多标签导致利用率被严重低估的问题）
- **#24093** / **#24095** - 修复 main 分支 lint/formatting 问题
- **#23700** - 统一 CI workflow 中 rust + protoc 安装
- **#24080** - 将 stage-b-test-1-gpu-large runner 池扩展到 H100 + H200

### 4. Apple Silicon / MLX 进展
- **#22416** - MLX decode 部分重叠调度（async eval），提升 Apple Silicon 生成性能

### 5. Speculative Decoding 规范化
- **#21062** - 默认启用 spec v2
- **#24094** - 添加 speculative decoding 命名规范规则
- **#24081** - 将 `accepted_drafts` 重命名为 `correct_drafts` 消除歧义
