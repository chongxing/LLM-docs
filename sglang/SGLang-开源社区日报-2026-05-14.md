## 🔍 技术趋势与洞察

### 1. DeepSeek-V4 生态全面铺开
- FlashInfer SM90 cutlass MXFP4 MoE backend (W4A16) 新增支持 GPT-OSS + DeepSeek-V4
- BF16 EP-MoE DeepGEMM 支持
- KV Compression V2 从 DSV4 dev branch 移植
- P2P KV cache transfer engine for PD disaggregation（支持 4090/5090 消费级显卡）
- DeepSeek-V4 w4a4 MegaMoE 推进中

### 2. PD 分离 (Disaggregation) 持续演进
- PD 分离模式下的优先级调度修复
- NIXL staging buffer 支持异构 TP KV 传输
- HiCache 重构 + O_DIRECT 支持
- gRPC 协议层增加 SubscribeKvEvents 流和 KV 事件消息

### 3. Apple Silicon / MLX 生态加速
- 新增 mlx_q4 / mlx_q8 实时量化（on-the-fly）
- 流式懒加载量化（~270× 更快，~4.5× KV 节省）
- 自动检测 MLX-format quantization_config

### 4. 性能与优化
- bench_serving 新增 MEM profile 活动
- DeepSeekV3 trtllm MoE 路径中移除冗余 FP32 upcast
- Req token-id 存储迁移到 array.array('q') 提升性能

### 5. 开源合规
- 大量从 vLLM 移植的文件新增 SPDX 版权头
