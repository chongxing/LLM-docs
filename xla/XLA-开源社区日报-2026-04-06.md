## 🔍 技术洞察

### 1. MemoryAllocator 重构
@ezhulenev 的 PR 39723 完成了重要的内存分配重构：
- 引入 `MemoryAllocator::AllocationTracker` 提供线程安全的内存追踪
- 将 CUDA 设备/主机/共享内存分配统一到新 API
- 移除旧的 `MemoryTracker`，实现更稳健的内存管理
- 展示了对集体内存分配的正确处理

### 2. SPIR-V 到模块级别迁移
@hanhanW 的工作将 SPIR-V pipeline 點移至 model scope：
- 进一步简化首端编译流程
- 改进 pass 接受 CLI GPUTarget 测试标志
- 增强对目标缺失情况的错误处理

### 3. Tokenizer 修复
@JornTuyls 的 PR 23805 修复了 `rstrip` 标志的处理：
- 正确将 `rstrip=true` 视为后处理指令而非匹配条件
- 修复了特殊 token 如 `<|user|>` 在特정情况下的子词切分问题
- 确保shape_cast 和 transpose 操作正确拉出循环
